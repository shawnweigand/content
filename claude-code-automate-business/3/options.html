Section 03  ·  Three Ways to Run

The further right
you go, the more
Claude can do.

Claude Code runs in three environments. Each unlocks more capability. Only one gives you the full picture.

01
Web
claude.ai

Zero setup. Open a browser and start. Best for quick tasks, one-off answers, and exploring what Claude can do — before committing to a workflow.

Zero setup required
Conversation history
No filesystem access
No MCP / tool calls
No agentic execution
02
Desktop
Claude app

Faster, with better session persistence and a native feel. Supports Projects and CLAUDE.md. Still limited — no bash, no subagents, no real autonomy.

Projects & CLAUDE.md
Better persistence
Native performance
No bash / CLI execution
No subagents
03
Terminal
claude code CLI

Full agentic power. Filesystem, bash, MCP servers, SKILL.md, subagents, hooks. Claude works autonomously on your machine like a real engineer — in a loop, until the job is done.

Full filesystem access
Bash & CLI execution
MCP servers & tools
Subagents & SKILL.md
Hooks & autonomous loops
